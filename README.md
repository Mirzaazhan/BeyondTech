# Beyond Horizon — User Directory

A User Directory app built as a technical assessment submission. Fetches 10 users from the JSONPlaceholder API and presents them with search, filtering, sorting, and detail views.

## Features

- Live search with inline highlight
- A–Z letter filter bar
- Sort by name, username, or city
- Grid / list view toggle
- User detail modal (email, phone, address, company, website)
- Favorites panel with localStorage persistence
- Light / dark theme toggle
- Page intro animation
- 3D card tilt on hover
- Skeleton loading + error state
- Keyboard shortcut: `/` to focus search
- Fully responsive, WCAG AA, respects `prefers-reduced-motion`

## Stack

Vanilla HTML, CSS, and JavaScript — no build step, no dependencies.
